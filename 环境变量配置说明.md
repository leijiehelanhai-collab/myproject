# 🔧 环境变量配置说明

## 📝 .env 文件配置

请在项目根目录的 `.env` 文件中添加以下配置：

```env
# 默认合约地址（最新部署的合约）
REACT_APP_CONTRACT_ADDRESS=0xbDb50858C1D86005a8570fd9dD3ebEaB1B08A5eD

# 测试代币地址
REACT_APP_TEST_TOKEN_1=0xC6c568F57296fBcD9d9BA07583198D2f267c4d50
REACT_APP_TEST_TOKEN_2=0x0c233c60853018483847DE9499eD506dfaa0AD4c

# 其他现有配置（请保留）
PRIVATE_KEY=你的私钥
DEV_WALLET=你的开发者钱包地址
BSC_TESTNET_URL=你的BSC测试网RPC地址
```

## 🎯 配置效果

### 1. 自动合约连接
- ✅ 连接钱包后自动显示默认合约地址
- ✅ 一键使用配置的合约地址
- ✅ 仍保留手动输入选项

### 2. 快捷代币选择
- ✅ 管理员面板显示"快捷选择"按钮
- ✅ 一键填入预配置的测试代币地址
- ✅ 提升开发和测试效率

## 🔄 更新合约地址

当部署新合约时，只需要：

1. **更新 .env 文件**：
   ```env
   REACT_APP_CONTRACT_ADDRESS=新的合约地址
   ```

2. **重启前端**：
   ```bash
   npm start
   ```

3. **自动生效** - 无需修改代码

## 📊 配置项说明

| 变量名 | 用途 | 必需 | 示例值 |
|--------|------|------|--------|
| `REACT_APP_CONTRACT_ADDRESS` | 默认合约地址 | 可选 | `0xbDb...A5eD` |
| `REACT_APP_TEST_TOKEN_1` | 第一个测试代币 | 可选 | `0xC6c...4d50` |
| `REACT_APP_TEST_TOKEN_2` | 第二个测试代币 | 可选 | `0x0c2...AD4c` |

## 🛠️ 开发者说明

### 添加新的测试代币
在 `.env` 中添加：
```env
REACT_APP_TEST_TOKEN_3=新代币地址
```

然后在代码中使用：
```javascript
process.env.REACT_APP_TEST_TOKEN_3
```

### 环境变量命名规则
- 必须以 `REACT_APP_` 开头（React 要求）
- 使用大写和下划线
- 描述性命名
